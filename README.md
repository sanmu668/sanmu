# AI 智能客服 SaaS 系统

这是一个基于 SpringBoot + Vue3 + MySQL 构建的智能客服系统，具备以下功能：

- 用户管理（注册 / 登录 / 修改 / 删除）
- 智能推荐（简历分析、岗位匹配）
- 职业路径规划（AI辅助）
- 后台管理系统（权限控制）

## 技术栈

- 后端：Spring Boot、MyBatis、JWT、MySQL
- 前端：Vue3、Element Plus、Axios
- 数据分析：Python（Pandas / scikit-learn / Flask）

## 快速开始

```bash
# 克隆项目
git clone https://github.com/你的用户名/你的项目名.git

# 进入项目目录
cd ai-customer

# 启动后端
cd backend
mvn spring-boot:run

# 启动前端
cd frontend
npm install
npm run dev
