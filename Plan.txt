《智简未来》-智能招聘系统
前台界面（求职者端）
页面			功能点
登录 /注册页	简洁表单登录（邮箱+密码 / 第三方登录选配）
首页（Dashboard）	展示用户最近简历状态、投递数、推荐职位卡片
简历管理页	个人简历填写模块（基本信息、教育经历、工作经历、技能、证书）支持上传附件（PDF简历）
职位推荐页	基于简历信息智能推荐岗位（列表卡片+分页）
投递记录页	查看历史投递列表（岗位名+公司+状态）
AI对话页    	可以与智能客服进行交互
账号设置页	修改头像、修改密码、设置通知等

后台界面（管理员/HR端）
页面		功能点
登录页	HR/Admin账户登录
用户管理	列表查看所有注册用户+支持封禁/修改权限
简历查看	查看某个用户完整简历（分模块展开）
职位管理	添加/编辑/删除招聘岗位，设定要求/截止时间等
投递管理	查看所有简历投递记录，筛选职位、状态（如已读/未读）
数据统计	关键指标仪表盘（用户数、岗位数、投递数、转化率等）

后端功能模块
模块		功能		核心接口
认证模块	登录注册、角色验证	/auth/*
（在安全性方面，结合了 Spring Security 和 JWT (JSON Web Token) 实现了用户认证和授权，使用 BCrypt 加密算法保护用户密码安全。数据持久化层使用了 Spring Data JPA 配合 MySQL 数据库，提供高效的数据访问。系统通过 @CrossOrigin 注解和安全配置解决了跨域问题，使用 JWT 无状态令牌解决了分布式系统的会话管理问题，同时通过统一的异常处理机制提供了友好的错误响应。）

简历管理	上传、编辑简历	/resume/*
职位推荐	推荐岗位	/job/recommend
投递记录	投递管理	/apply/*
账号设置	头像、密码、通知设置	/user/*
AI对话功能                        /chat/*
（AI简历优化与职业咨询服务，核心使用阿里云DashScope的通义千问(Qwen)大语言模型作为AI对话引擎，通过RESTful API提供服务。系统包含对话管理、历史记录存储、安全控制等功能模块，采用MySQL数据库持久化数据，并通过Spring Security实现接口访问控制。）

用户管理（后台）	管理求职者	/admin/users
简历查看（后台）	查看任意求职者简历	/admin/resume/*
职位管理（后台）	岗位增删改查	/admin/job/*
SpringData JPA通过事务管理+实体状态跟踪（由 Hibernate 实现）+增删查改方法，将对对象的修改自动同步到数据库中。
实现动态查询功能，使用 Specification<Job> 构建灵活的查询条件（如职位名、公司名、状态等），支持分页与模糊搜索。
完成前后端接口联调，使用 JSON 格式数据通信，实现数据的动态渲染与实时更新；
